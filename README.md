# ⬇️ Instalação

[Termux](https://play.google.com/store/apps/details?id=com.termux&hl=pt_BR&gl=US)

```
git clone https://github.com/ianmsfvenom/Brizas-bot
cd Rigby-bot
bash install.sh

(iniciar o bot manualmente)
npm start

(reiniciar o bot automaticamente em caso de crash)
sh start.sh

(ou)
pm2 start index.js

```
